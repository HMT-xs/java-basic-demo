项目简介
面向对象基础进阶练习，聚焦类、封装、继承，为多态与抽象类学习做铺垫。
 
文件结构
2026-3-12-java/
├── Person.java      # 父类：人类（基础属性与方法）
├── Student.java     # 子类：学生类（继承 Person，封装学生信息）
├── Animal.java      # 抽象动物类（定义抽象方法）
├── Dog.java         # 子类：狗类（实现 Animal 抽象方法）
├── Cat.java         # 子类：猫类（实现 Animal 抽象方法）
├── Phone.java       # 父类：手机类
├── SmartPhone.java  # 子类：智能手机类（继承 Phone）
├── Car.java         # 汽车类（独立类练习）
├── Employee.java    # 抽象员工类
├── Manager.java     # 子类：经理类（实现 Employee 抽象方法）
└── Tesal.java       # 综合测试入口

运行说明
直接运行对应.java文件即可
