项目简介
这是一份 Java 入门阶段 的练习代码集合，包含数组操作、循环控制、基础算法等核心知识点，适合新手巩固语法与逻辑。
 
代码分类说明
1. 基础变量与交换类

ArrTest1.java ：演示两数交换原理，学习临时变量在变量值交换中的使用；同时实现数组首尾元素交换，理解数组下标与元素操作。
ArrTest2.java ：使用双指针法反转数组，遍历并输出反转后的结果，强化数组遍历与交换逻辑。
ArrTest3.java ：利用  Random  类随机打乱数组元素，实现数组元素的随机重排，练习随机数生成与数组元素交换。
 
2. 数组操作类

Exercise1.java ：创建长度为6的数组，生成0-99的随机数填充数组，计算数组元素总和并遍历输出。
Exercise2.java ：遍历数组并在同一行输出所有元素，练习数组遍历与控制台输出格式控制。
Exercise3.java ：查找数组中的最小值，通过遍历比较更新极值，实现基础极值查找逻辑。
Exercise4.java ：将数组元素拼接为整数（如  {1,2,3,4,5}  →  12345 ），强化数组遍历与数值运算逻辑。
Exercise5.java ：计算数组元素的平均值，练习数组求和与浮点型除法运算。
Exercise6.java ：接收用户输入的整数，将其按顺序插入到已排序数组中，实现数组元素插入与排序逻辑。
Exercise7.java ：将数组中奇数放在左侧、偶数放在右侧，使用双指针（左存奇、右存偶）实现元素分类。
 
运行说明

直接运行对应  .java  文件即可。
