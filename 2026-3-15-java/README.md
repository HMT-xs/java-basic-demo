项目简介
这是一份 Java 入门阶段的练习代码集合，包含数组操作、循环控制、基础算法等核心知识点，适合新手巩固语法与逻辑。
 
代码分类说明
 
1. 数组操作类
ArrayDemo1.java：数组遍历、求和、统计可被3整除的元素、偶数元素修改（乘2/除以2）
ArrayDemo2.java：查找数组中的最大值
ArrayDemo3.java：生成随机数组，计算总和、平均值，并统计小于平均值的元素个数
 
2. 循环控制类
LoopTest1.java：逢七过游戏（跳过含7或7的倍数的数字）
LoopTest2.java：计算整数的平方根或其整数部分
LoopTest3.java：判断一个正整数是否为质数
LoopTest4.java：猜数字小游戏（最多5次机会）
SkipLoopDemo1.java：演示  continue （跳过某次循环）和  break （结束整个循环）的区别
 
3. 业务逻辑类
Test7.java：根据税前工资计算个人所得税及税后工资（模拟三险一金扣除与阶梯税率）

运行说明
直接运行对应.java文件即可
