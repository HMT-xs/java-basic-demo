项目简介
 
这是一份 Java 入门阶段的练习代码集合，包含条件分支、随机生成、数组处理、加密解密、综合业务逻辑等核心知识点，适合新手巩固语法与逻辑。
 
 
代码分类说明
 
1、业务逻辑与条件判断类
Test1.java：实现机票价格计算功能，根据出行月份与舱型，通过多层条件分支计算不同折扣后的实际票价。
 
2、随机生成与字符处理类
Test2.java：随机生成五位验证码，前四位为大小写字母，最后一位为数字，练习 ASCII 码转换与随机字符拼接逻辑。

3、数组与数据统计类
Test3.java：实现评委打分求平均分功能，封装分数录入、求最大值、求最小值的方法，练习数组遍历与极值查找逻辑。

4、加密与解密算法类
Test4.java：实现密码加密逻辑，通过“加5取余+数组反转+拼接”三步操作，完成数字密码的加密传输。
Test6.java：实现加密密码的解密逻辑，与 Test4 对应，通过“反转+补位+减5”三步操作还原原始密码。

5、数字与数组转换类
Test5.java：实现将整数每一位拆分到数组的功能，先计算位数、再动态初始化数组，最后逆序存入每一位数字，强化循环与数组索引操作。

6、综合业务与游戏类
Test7.java：实现双色球彩票中奖判断功能，包含生成中奖号码、录入用户购买号码、统计红球与蓝球匹配数等完整流程，练习方法封装与嵌套循环逻辑。


运行说明
 
直接运行对应  .java  文件即可。
