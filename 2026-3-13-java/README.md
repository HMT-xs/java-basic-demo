项目简介
 多态与抽象类专项练习，通过多场景案例巩固核心概念，为 SpringBoot 开发打基础。
 
文件结构
2026-3-13-java/
├── Animal.java          # 抽象动物类
├── AnimalTest.java      # 动物多态测试入口
├── Dog.java             # 狗类（继承 Animal）
├── Cat.java             # 猫类（继承 Animal）
├── Machine.java         # 抽象机器类
├── MachineTest.java     # 机器多态测试入口
├── Car.java             # 汽车类（继承 Machine）
├── Bike.java            # 自行车类（继承 Machine）
├── Shape.java           # 图形父类
├── ShapeTest.java       # 图形多态测试入口
├── Circle.java          # 圆形类（继承 Shape）
├── Rectangle.java       # 矩形类（继承 Shape）
├── Employee.java        # 抽象员工类
├── EmployeeTest.java    # 员工多态测试入口
├── Developer.java       # 开发人员类（继承 Employee）
├── Phone.java           # 手机父类
├── PhoneTest.java       # 手机多态测试入口
└── SmartPhone.java      # 智能手机类（继承 Phone）

运行说明
直接运行对应.java文件即可
