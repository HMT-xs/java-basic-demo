项目简介

这是一份 Java 入门阶段的练习代码集合，包含方法封装、数组操作、循环控制、基础算法等核心知识点，适合新手巩固语法与逻辑。
 

代码分类说明

1、基础方法与工具类
MethodTest1.java：定义遍历数组的方法，学习方法封装与数组格式化输出逻辑，实现带逗号分隔的数组打印。
MethodTest2.java：封装求数组最大值的方法，掌握方法返回值与遍历查找极值的实现。
MethodTest3.java：实现判断数组是否包含指定元素的方法，练习布尔型返回值与循环查找逻辑。
MethodTest4.java：编写数组范围拷贝方法，学习伪索引思想与指定区间的数组截取操作。
 
2、数值运算与工具类
Exercise1.java：接收用户输入的三个整数，封装求最大值、最小值的方法，结合三元运算符实现数值比较与中间值计算。
Exercise2.java：自定义求绝对值的方法，练习条件判断与浮点型数据处理。
Exercise3.java：实现计算正整数位数的方法，通过循环除10的方式统计数字长度，强化循环与变量计数逻辑。
 
3、数组操作类
Exercise4.java：封装数组相等判断方法，先比较长度再逐元素对比，学习数组遍历与布尔返回逻辑。
Exercise5.java：实现数组填充方法，将数组所有元素替换为指定值，练习数组遍历与元素赋值。
Exercise6.java：编写数组拷贝方法，实现指定长度的数组截取，掌握新数组创建与数据迁移逻辑。
 
4、综合应用与算法类
Exercise7.java：实现抽奖系统，结合 Random 类与自定义方法，完成不重复抽奖的逻辑，练习随机数生成与数组去重判断。
Exercise8.java：对抽奖系统进行优化，使用 Fisher-Yates 洗牌算法打乱奖池数组，实现更高效的随机遍历输出。
 

运行说明
 
直接运行对应  .java  文件即可。
